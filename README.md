## ✨ Возможности

### 🎯 Основные функции
- **Drag & Drop** — перетаскивание файлов в обе стороны
- **Иерархическая структура** — папки и подпапки любой глубины
- **Автооткрытие папок** — бросайте файлы прямо в открытые папки
- **Просмотр файлов** — двойной клик открывает файл в системном приложении
- **Экспорт всех файлов** — одной кнопкой с сохранением структуры
- **Физическое удаление** — .enc файлы удаляются с диска при удалении из сейфа

### 🛡️ Защита от брутфорса
- **10 попыток** до самоуничтожения
- **Crypto-shredding** — уничтожение master-ключа (данные невосстановимы)
- **Экспоненциальные задержки** — 2с → 3с → 5с → 8с → 12с → 18с
- **Tamper-evident счётчик** — защищён HMAC-подписью
- **Автогенерация** 50-символьного криптографически стойкого пароля

### 🔒 Криптография военного уровня
- **AES-256-GCM** — шифрование файлов (стандарт TLS 1.3)
- **Argon2id** — KDF с параметрами memory=128MB, time=4
- **Двухуровневая архитектура** — Master Key + KEK
- **Уникальные nonce** для каждого файла
- **Зашифрованный индекс** — структура сейфа недоступна без пароля

### 🎨 Интерфейс
- Тёмная тема в стиле Material Design 3
- Визуальная подсветка папок при drag & drop
- Placeholder для пустых папок (70px зона для drop)
- Контекстное меню (ПКМ) для всех операций
- Кнопки "Открыть/Закрыть" для папок

## 🔐 Безопасность

### Вероятность взлома

| Вектор атаки | Вероятность успеха |
|---|---|
| Чистый криптоанализ AES-256-GCM | ≈ 0% (невозможно) |
| Атака на слабый пароль (8 символов) | 80-95% при GPU-ферме |
| Атака на сильный пароль (16+ символов) | < 0.001% |
| Брутфорс (с защитой программы) | **0%** (самоуничтожение после 10 попыток) |
| Компрометация через malware/keylogger | 70-90% (проблема ОС, не программы) |

### Что защищает
✅ Криптографическая стойкость AES-256-GCM  
✅ Защита от GPU-ферм через Argon2id (128 MB памяти)  
✅ Невозможность восстановления после самоуничтожения  
✅ Tamper-evident защита счётчика попыток  
✅ Полная автономность (не требует интернета)  

### Что НЕ защищает
⚠️ Слабые пароли (< 10 символов)  
⚠️ Keyloggers и malware на вашем устройстве  
⚠️ Физический доступ к RAM (cold boot attack)  
⚠️ Скриншоты экрана  

### Рекомендации
1. **Используйте сгенерированный пароль** (50 символов)
2. **Сохраните пароль** в надёжном месте (менеджер паролей)
3. **Не используйте** на заражённых системах
4. **Делайте бэкапы** зашифрованной папки `secure_vault_data`

## 🚀 Установка

### Способ 1: Из исходников (рекомендуется)

```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/secure-vault-pro.git
cd secure-vault-pro

# Создайте виртуальное окружение
python -m venv .venv

# Активируйте
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# Установите зависимости
pip install -r requirements.txt

# Запустите
python safe.py
```

### Способ 2: Быстрая установка

```bash
pip install PySide6 cryptography argon2-cffi
python safe.py
```

### Способ 3: Готовый .exe

Скачайте последнюю версию из [Releases]([https://github.com/YOUR_USERNAME/secure-vault-pro/releases](https://github.com/Python-k19/Secure-Vault-Pro/releases/tag/Secure-Vault-Pro))

## 📖 Использование

### Первый запуск

1. Запустите программу
2. Создастся папка `secure_vault_data` рядом с программой
3. Нажмите **"Использовать этот"** для 50-символьного пароля
4. Подтвердите пароль
5. Нажмите **"ОТКРЫТЬ СЕЙФ"**

### Работа с файлами

**Добавление файлов:**
- Перетащите в верхнюю зону → попадёт в корень
- Перетащите на открытую папку → попадёт внутрь папки
- Перетащите на placeholder пустой папки → попадёт внутрь

**Просмотр:**
- Двойной клик по файлу → откроется в системном приложении
- ПКМ → "Открыть для просмотра"

**Экспорт:**
- Перетащите файлы из сейфа на рабочий стол
- Кнопка **"📦 Извлечь все"** → выбрать папку
- ПКМ → "Экспортировать..."

**Удаление:**
- Выберите файлы → кнопка **"🗑 Удалить выбранное"**
- ПКМ → "Удалить"
- Файлы удаляются с диска физически

### Создание папок

- Кнопка **"📁 Новая папка"** в тулбаре
- ПКМ на папке → "Создать подпапку"
- ПКМ на пустом месте → "Создать папку здесь"

## 🏗️ Архитектура

### Структура данных

```
secure_vault_data/
├── .salt                      # Соль для Argon2id
├── .master_key.enc           # Зашифрованный master key
├── .vault_meta.enc           # Зашифрованный индекс файлов
├── .vault_attempts           # Счётчик попыток (HMAC)
└── *.enc                     # Зашифрованные файлы
```

### Двухуровневая криптосхема

```
Пароль
   ↓
Argon2id (128MB, 4 итерации)
   ↓
KEK (Key Encryption Key)
   ↓
Расшифровка Master Key
   ↓
Master Key (256 бит, случайный)
   ↓
AES-256-GCM шифрует файлы
```

### Crypto-Shredding

При 10-й неудачной попытке:
1. Master Key перезаписывается случайными данными 7 раз
2. Master Key удаляется с диска
3. Соль удаляется с диска
4. **Результат:** файлы невозможно расшифровать даже при знании пароля

## 📦 Сборка в .exe

### Автоматическая сборка

```bash
python build.py
```

Скрипт автоматически:
1. Создаст виртуальное окружение
2. Установит все зависимости
3. Скомпилирует .exe через PyInstaller
4. Откроет папку с результатом

### Ручная сборка

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=1.ico --collect-all=PySide6 safe.py
```

**Результат:** `dist/SecureVault.exe` (~150-200 MB)
